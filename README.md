# AI Website Chatbot

## 📌 Overview
This project is an AI-powered chatbot integrated into a website to simulate customer support for a local business.

## 🚀 Features
- Answers customer questions using AI
- Simple web interface (HTML + JavaScript)
- Backend API built with Flask
- Real-time communication between frontend and backend

## 🛠️ Tech Stack
- Python (Flask)
- OpenAI API
- HTML, CSS, JavaScript

## 📷 Demo
![Chatbot Demo](https://i.imgur.com/JGl2qB1.png)

## ⚙️ How to Run
1. Clone the repository
2. Create virtual environment:
   python -m venv venv

3. Activate it:
   venv\Scripts\activate

4. Install dependencies:
   pip install flask openai flask-cors

5. Run the backend:
   python app.py

6. Open index.html in browser

## 💡 Use Case
This chatbot can help small businesses automate responses to common customer queries such as pricing, location, and availability.
